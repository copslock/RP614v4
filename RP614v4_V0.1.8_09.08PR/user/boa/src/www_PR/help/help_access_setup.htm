无线网卡访问设置帮助

可用的无线网卡

可用的无线网卡列表显示了所有可用的无线PC网卡及其MAC地址。

对于可用的无线网卡列表中显示的无线PC网卡，单击其单选按钮即可捕获其MAC地址。 列表中没有显示所需的无线PC网卡时，首先确认该网卡配置正确无误，然后单击 刷新 按钮来更新可用无线PC网卡的列表。如此时仍未显示所需的无线PC网卡，则请按以下说明手工设置其MAC地址。

输入无线网卡

可用的无线网卡列表中未显示任何无线网卡时，可手工输入待授权无线PC网卡的“设备名称”和“MAC地址”。

注意：MAC地址是一个字符串，其中包含十二个字符，通常可在无线设备的底部找到。

添加或者取消更改

设置更改完毕后，请单击添加 。
单击取消即可恢复到原来的设置。
单击刷新可显示当前已接设备。
