服务限制设置帮助

利用服务限制功能，可按照本地网络中具体用户的IP地址，限制这些用户对因特网的访问。 此外还可以彻底阻止用户使用某些因特网服务。

新增服务

  1. 从下拉列表中选择服务类型或“用户定义”（列表中没有所需服务类型时）。
  2. 对于“用户定义”，必须选择协议，并输入服务使用的名称和端口范围。 对于已知服务，这些字段将被自动填入。
  3. 设置IP地址选项，确定将被限制的PC网卡。 （详情如下）。
  4. 单击应用保存更改。

对以下IP地址的服务进行过滤——这决定了哪些计算机会被限制。

  • 只有此IP地址——仅对一块PC网卡进行限制。 输入将被限制的PC网卡的IP地址。
  • IP地址范围——将要限制一组PC网卡（由IP地址决定）。 输入将被限制的PC网卡的IP地址范围（起始地址和结束地址）。
  • 所有IP地址——所有PC网卡都将被限制。

保存或者取消更改

服务限制表更新完毕后，单击添加。
单击取消即可返回到此前设置。
