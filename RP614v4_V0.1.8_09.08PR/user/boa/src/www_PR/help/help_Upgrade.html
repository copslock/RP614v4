路由器升级帮助

通过路由器升级页面可以安装路由器的新版本软件。

请访问NETGEAR网站，找到路由器新版软件。 升级文件下载完毕后需要对其进行解压，然后方可升级路由器。

要点！点击升级 之后切勿中断软件到路由器的发送以及路由器的重启过程。 如果认为该过程可能会以某种方式中断，则可单击取消，以保留当前的路由器软件。

要升级路由器软件：

  1. 访问www.NETGEAR.com，下载升级软件。
  2. 如果下载的软件没有自动解压，则将其解压。
    在继续以后操作之前，可以首先阅读版本说明。
  3. 单击浏览。
  4. 找到并选中刚刚下载并解压的文件。
  5. 单击升级，将该软件发送到路由器。
    这样新软件将被载入路由器，并令路由器重新启动。
    注意：路由器重启完成之前切勿对路由器进行任何操作，如联机、关闭路由器、关闭计算机等等！ 等测试灯停止闪烁，几秒钟之后再对其进行操作。
  6. 单击路由器状态，检查固件版本，确认路由器现在已经安装了新软件。

要点！ 某些情况下（比如重大升级时），可能需要在升级之后重新配置路由器。 请参阅软件中附带的版本说明，了解是否需要重新配置路由器。

如果采用本方法无法成功升级，则请参阅路由器资源CD光盘上的参考手册，查阅路由器的其它升级方法。
