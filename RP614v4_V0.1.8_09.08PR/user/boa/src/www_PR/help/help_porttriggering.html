端口转发/端口触发帮助

端口触发是一种先进的功能，可用于游戏和其它因特网应用。 端口转发一般也能实现类似的功能，但它是静态的，并且有一定局限性。

比如说，端口触发会临时打开一个输入端口，不要求因特网上的服务器来跟踪DHCP是否改变了你的IP地址。

端口触发监视出站流量。 当路由器在某个指定的出站端口上检测到流量时，会记住那个发送数据并且“触发”输入端口的计算机的IP地址。 被触发端口上的输入流量此后将被转发到这台触发计算机。

使用端口转发/端口触发页面，可实现从因特网上对本地计算机或服务器的访问，从而实现各种服务（例如FTP或HTTP）、运行因特网游戏（如Quake II），或者使用因特网应用功能（如CUseeMe）。

端口转发用于 FTP、Web服务器或其它基于服务器的服务。 设置了端口转发之后，来自因特网的请求将被转发到适当的服务器。 与之相反，只有“触发了”指定的端口之后，端口触发才会允许来自因特网的请求。 端口触发应用于聊天和因特网游戏。

端口触发

注意： 选中了禁用端口触发复选框之后，所有端口触发功能都将被禁用，但配置信息仍在路由器中保留。

端口触发超时

请输入一个不大于9999分钟的数值。 端口触发超时值所控制的,就是指定的入站端口的无活动定时器。 无活动定时器超时后，入站端口将被关闭。

因特网游戏或者应用

开始前,首先要了解待配置的是哪种服务、应用或者游戏。 此外还需了解该游戏或者应用的出站端口（触发端口）地址。

要设置某台计算机，以运行因特网游戏或者使用因特网应用功能：

  1. 点击 添加服务 按钮。
  2. 在服务名称框中输入服务名称。
  3. 在服务用户下面可以选择任何人（缺省），允许用户网络中所有人使用本项服务。 而选择单一地址时，还需输入计算机的IP地址，从而把本项服务限定到这台特定的计算机。
  4. 选择服务类型。
  5. 在触发端口框内输入出站端口号。
  6. 输入入站连接端口信息，例如连接类型、起始端口、结束端口等。 此类信息可从游戏或者应用的手册中或者支持网站上找到。
  7. 单击应用按钮。

要修改某项服务：

  1. 选中表内该项服务的单选按钮。
  2. 单击编辑服务。
  3. 修改服务。
  4. 单击应用按钮。

要删除某项服务：

  1. 选中表内该项服务的单选按钮。
  2. 单击删除服务按钮。
