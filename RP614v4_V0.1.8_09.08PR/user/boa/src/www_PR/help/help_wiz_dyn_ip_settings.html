动态IP设置帮助

帐户名

（也就是通常所说的主机名或系统名）

对于多数用户，只需在此框内输入帐户名或用户名即可。 例如，主邮件帐号为JerAB@ISP.com时，请在框内输入JerAB。

如果ISP提供有特定的主机名，那么就输入该主机名（例如，CCA7324-A）。

域名

本框对多数用户来说都是不填的，但ISP要求填写时除外。 需要填写时，请填写ISP的域名。 例如，若ISP的邮件服务器为mail. xxx.yyy.zzz，则输入xxx.yyy.zzz作为域名。

如果ISP提供了域名，那么请在本框内填写这个域名。 （例如，Earthlink Cable可能会要求主机名为‘home’，而Comcast有时会提供域名。）

使用电缆调制解调器时，域名通常为工作组名。

域名服务器地址

域名服务器用于通过网站名称查找网站地址。

ISP提供了一到两个域名服务器地址时，请选择使用下面的域名服务器地址，然后输入首选和备用域名服务器的地址。

其它情况下请选择从ISP处动态获取。

注意：如果在访问网站时发生了“Address not found（地址未找到）”错误，那么很可能是域名服务器设置不当。 此时请联系ISP，索要域名服务器地址。

路由器MAC地址

计算机的本地地址在用户网络上是唯一的。 该地址也称为计算机的MAC（介质访问控制）地址。

通常情况下请选择使用缺省MAC地址。

如果ISP要求MAC认证，则请选择使用计算机MAC地址或者使用下面的MAC地址。前一种情况下将把当前计算机的MAC地址伪装成路由器的MAC地址，后一种情况下则需手动输入其它计算机的MAC地址。

MAC地址格式为XX:XX:XX:XX:XX:XX。 在选择了使用下面的MAC地址并且相应地设定了MAC值后，如果再选择使用计算机MAC地址，就会覆盖掉所设定的MAC值。

应用、取消和测试

如果您认为设置正确，或者在设置更改完毕之后，请单击应用。

单击取消即可恢复到缺省设置。

单击测试可连接到NETGEAR网站。 如果连接成功，则表明设置正常。此后可单击注销退出这些页面，然后……尽情享受网上冲浪带来的乐趣！

如果连接失败，

  • 联系ISP,核实配置信息。
  • 阅读 安装指南中的《故障处理》一节。
  • 阅读 资源CD光盘中的故障处理指南，或者参考手册中《故障处理》一节。
  • 联系NETGEAR技术支持部门。
