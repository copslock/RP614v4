固定IP服务帮助

因特网IP地址

如果拥有固定（或静态IP）地址，则ISP肯定已经提供了必需的信息。 将IP地址、子网掩码和网关IP地址键入正确的框内。

例如：

  • IP地址: 24.218.156.183
  • 子网掩码: 255.255.255.0
  • 网关IP地址: 24.218.156.1

域名服务器地址

域名服务器用于通过网站名称查找网站地址。

ISP提供了一到两个域名服务器地址时，请选择使用下面的域名服务器地址，然后输入首选和备用域名服务器的地址。

其它情况下请选择从ISP处动态获取。

注意：如果在访问网站时发生了“Address not found（地址未找到）”错误，那么很可能是域名服务器设置不当。 此时请联系ISP，索要域名服务器地址。

应用、取消和测试

设置更改完毕后，请单击应用 。

单击取消即可恢复到原来的设置。

单击测试可连接到NETGEAR网站。 如果连接成功，则表明设置正常。此后可单击注销退出这些页面，然后……尽情享受网上冲浪带来的乐趣！

如果连接失败，

  • 则请检查设置，确保所做选择及每一项输入都正确无误。
  • 联系ISP,核实配置信息。
  • 阅读 安装指南中的《故障处理》一节。
  • 阅读 资源CD光盘中的故障处理指南，或者阅读参考手册中《故障处理》一节。
  • 联系NETGEAR技术支持部门。
