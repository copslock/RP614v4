备份设置帮助

通过本页面可以备份、恢复和擦除路由器的当前设置。

路由器工作正常后应备份其信息，以备故障时使用。 备份设置时，备份将以文件的形式保存到计算机上。 用户可从该文件中对路由器设置进行恢复。

要创建当前设置的备份文件：

  1. 单击备份。
  2. 如果浏览器尚未设置成自动保存下载的文件，则应指定文件保存路径，并根据需要对文件进行重命名，然后单击保存。
    如果浏览器已设置成自动保存下载的文件，则文件会保存到硬盘上的浏览器默认下载路径，所用文件名为netgear.cfg。

要点！在开始恢复或者擦除路由器设置后、路由器完成重启前，切勿对其进行任何操作，如联机、关闭路由器、关闭计算机等等！ 路由器设置的擦除与恢复应该只需一分钟左右。 等测试灯停止闪烁，几秒钟之后再对其进行操作。

要从备份文件中恢复设置：

  1. 单击浏览。
  2. 找到并选中此前保存的备份文件（缺省为 netgear.cfg)。
  3. 单击恢复。
    此时会弹出一个窗口通知您路由器已成功恢复到此前的设置。 此后路由器将重新启动。 时间大概需要一分钟。
    要点！路由器重启完成之前切勿对其进行任何操作，如联机、关闭路由器、关闭计算机等等！ 另外测试灯停止闪烁后，要再等几秒钟，然后再操作路由器。
  4. 关闭消息窗口。

擦除当前设置，并将路由器复位到原始出厂缺省设置：

  • 单击擦除。
    要点！路由器重启完成之前切勿对其进行任何操作，如联机、关闭路由器、关闭计算机等等！ 另外测试灯停止闪烁后，要再等几秒钟，然后再操作路由器。

擦除了路由器的当前设置之后，路由器密码为password，局域网IP地址为192.168.1.1；在局域网上，它是DHCP服务器，在因特网上则是DHCP客户端。
