服务限制帮助

利用服务限制功能，可按照本地网络中具体用户的IP地址，限制这些用户对因特网的访问。 比如说，可以禁止用户网络上的所有人或者某些人使用某些因特网服务，例如网上冲浪（http）。 服务限制可配置成从不、按时间安排或始终。

要启用服务限制功能：

    单击按时间安排打开服务限制功能，并按照时间安排页面进行限制。
    单击始终打开服务限制功能，并一直启用，与时间安排页面无关。

要禁用服务限制：

    单击从不，关闭服务限制功能。

要新增服务

  1. 单击添加按钮。
  2. 从下拉框中选择一个缺省服务，或这输入新服务的所需信息。
  3. 完成后单击第二个添加按钮。

要编辑某服务

  1. 选择待编辑服务的单选按钮。
  2. 单击编辑按钮。
  3. 按需要更改服务的数据。
  4. 完成后单击接受按钮。

要删除某服务

  1. 单击表内某行后边的单选按钮。
  2. 单击删除按钮删除所选服务。

保存或者取消更改

单击应用即可将新设置保存到路由器。
单击取消即可放弃所有尚未保存的更改。
