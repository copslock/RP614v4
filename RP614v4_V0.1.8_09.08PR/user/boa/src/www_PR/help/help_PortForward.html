端口转发/端口触发帮助

端口触发是一种先进的功能，可用于游戏和其它因特网应用。 端口转发一般也能实现类似的功能，但它是静态的，并且有一定局限性。

比如说，端口触发会临时打开一个输入端口，不要求因特网上的服务器来跟踪DHCP是否改变了你的IP地址。

端口触发监视出站流量。 当路由器在某个指定的出站端口上检测到流量时，会记住那个发送数据并且“触发”输入端口的计算机的IP地址。 被触发端口上的输入流量此后将被转发到这台触发计算机。

使用端口转发/端口触发页面，可实现从因特网上对本地计算机或服务器的访问，从而实现各种服务（例如FTP或HTTP）、运行因特网游戏（如Quake II），或者使用因特网应用功能（如CUseeMe）。

端口转发用于 FTP、Web服务器或其它基于服务器的服务。 设置了端口转发之后，来自因特网的请求将被转发到适当的服务器。

与之相反，只有“触发了”指定的端口之后，端口触发才会允许来自因特网的请求。 端口触发应用于聊天和因特网游戏。

端口转发

对于下拉菜单中业已存在的服务、应用或者游戏来说，只需规定计算机的IP地址即可。 在其它情况下则需单击添加自定义服务按钮，然后规定各项服务、游戏或者应用的端口号和计算机 IP 地址。

端口分配

您最多可针对因特网服务、应用或者程序进行20个不同的端口分配。 在“服务名称列表”中可以选择一个服务、应用或者游戏。 如果在所有列表中都没发现所需的项目，则请联系软件或者游戏开发人员，查找需要使用的正确端口号。

因特网服务

在开始之前，首先应了解即将提供的服务类型以及即将提供此类服务的计算机IP地址。 对于所提供的服务来说，最常见的就是Web（HTTP）服务或者FTP服务。

请按照以下步骤操作，设置一台可从因特网访问的计算机或者服务器以提供某种因特网服务：

  1. 从“服务名称”列表中选择一项需要使用的服务。
  2. 在服务器IP地址框中键入该计算机的IP地址。
  3. 单击添加。

注意： 可令一台计算机或者服务器提供多种类型的服务。 此时可再选择一个服务，然后为这台计算机或者服务器键入相同的IP地址。

因特网游戏或者应用

开始前,首先要了解待配置的是哪种服务、应用或者游戏。 此外还应了解需要使用的计算机IP地址。

要设置某台计算机，以运行因特网游戏或者使用因特网应用功能：

  1. 从相关的某个列表中选择需要使用的因特网应用或者游戏。
    此时起始端口和结束端口框中均会填充相应值。
    注意： 如果在清单中没有找到所需的游戏或者应用，则请单击添加自定义服务按钮，然后键入服务名称、起始端口、终止端口和服务器IP地址等信息。
  2. 在服务器IP地址框中键入该计算机的IP地址。
  3. 单击添加。

要再设置一台计算机来运行因特网游戏（比如Hexen II或者KALI）：

  1. 单击添加自定义服务按钮。
  2. 在服务名称框中键入服务名称。
  3. 在起始端口框中键入起始端口号。
    对于此类游戏来说，请使用默认列表中给出的端口号，每增加一台计算机，端口号加1。 举例来说，假设有一台计算机被配置成了采用端口26900来运行Hexen II，那么第二台计算机的端口号就应该是26901，第三台计算机的应为26902。
  4. 在结束端口框中键入相同端口号。
  5. 在服务器IP地址框中键入该计算机的IP地址。
  6. 单击应用。

要修改某项服务：

  1. 选中表内该项服务的单选按钮。
  2. 单击编辑服务。
  3. 从服务名称、起始端口、结束端口和服务器IP地址各框中修改信息。
  4. 单击应用。

要删除某项服务：

  1. 选中表内该项服务的单选按钮。
  2. 单击删除服务。
