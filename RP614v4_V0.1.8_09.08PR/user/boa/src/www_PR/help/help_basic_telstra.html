帮助

通过 设置页可对NETGEAR电缆/DSL路由器进行配置、升级和状态检查。

单击最左列中的某一项， 中间列即显示该区域的当前设置或者信息。

本列显示所选设置页的相关帮助信息。 使用Internet Explorer时，单击中间列某一项即可直接跳到相关的帮助章节；其它情况下请向下滚动到所需章节。

基本设置帮助

注意：首次进行路由器设置时，可能无需更改缺省设置即可让路由器正常工作。

您的因特网连接需要登录吗?

请根据您在ISP那里开通的帐号类型来选择本项。 如果每次连接因特网都要输入登录信息，或者在ISP那里开通的是PPPoE帐号，那么请选择是， 否则请选择否。

注意： 如果已经安装了PPP软件，如Earthlink的WinPoET、PacBell的Enternet，或者AOL的宽带软件，则拥有PPPoE帐户。 此时请选择“是”。

因特网服务提供商

ISP位于下拉菜单中时，选择该ISP；否则请选择其它。

AOL宽带DSL或电缆

选中了AOL电缆或AOL DSL时，路由器将自动配置AOL宽带连接所需的信息。 从连在路由器上的每台PC机访问因特网时，都要使用AOL提供的AOL Sign On客户端软件，否则根本无法上网。

注意： 请注意，使用“L2TP隧道”协议的应用程序（比如AOL）将绕过本路由器附带的防火墙。 为进一步保障家庭网络安全，建议安装基于PC的防火墙应用软件。

连接因特网：

要想连接AOL和因特网，只需按正常方式登录AOL服务即可。 唯一需要更改的就是登录屏幕上的选择位置： 设置，设置方式如下：

AOL DSL用户的位置应选作家庭网络。
AOL电缆用户的位置应选作家庭——电缆。

下拉框中未显示家庭网络或家庭——电缆选项时，需要由用户添加此类位置。 请参照NETGEAR路由器包装盒中附带的AOL文档，了解如何正确设置位置。 如果包装盒中没有这些文档，那么请按照如下说明操作：

AOL DSL用户

如果能登录AOL：
1. 登录AOL
2. 打开关键字： Home Network Setup（家庭网络设置）
3. 按屏幕说明操作
4. 注销AOL
5. 在登录屏幕上将位置选作“家庭网络”

AOL电缆用户

1. 在登录屏幕上将位置选作“家庭——电缆”

注意： 如果在访问网站时发生“Address not found（地址未找到）”错误，则很可能是您没有登录AOL或者把客户端设置成“家庭网络”。 如果问题依然存在，那么请参阅AOL文档，查找正确的设置步骤。

登录

通常为电子邮件地址中使用的用户名。 例如，主邮件帐号为JerAB@ISP.com时，请在框内输入JerAB。

某些ISP（如Mindspring 和 Earthlink）要求用户在登录时使用完整的电子邮件地址，此时请在登录框内输入完整的电子邮件地址。

AOL宽带用户无需在路由器上更改这些设置——这些设置均由NETGEAR自动输入。

密码

（不需要登录时不可用。）

请输入登录ISP时所用的密码。

AOL宽带用户无需在路由器上更改这些设置——这些设置均由NETGEAR自动输入。

服务名称

对于多数用户来说，都用不着填写本框。

如果ISP提供了帐户名，那么请在本框内填写这个帐户名。 （例如，某些Earthlink帐户可能会要求服务名称为@earthlink.net。）

闲置超时

（不需要登录时不可用。）

路由器将用户从服务器断开之前所允许的闲置时间（分钟）。 建议的缺省值为5。

认证服务器

缺省名称为sm-server。 当地ISP可能会随时更改认证名称。尽管这种情况通常都发生得不太频繁，但一旦发生，可能就需要更新认证服务器。

域名服务器地址

域名服务器用于通过网站名称查找网站地址。

ISP提供了一到两个域名服务器地址时，请选择使用下面的域名服务器地址，然后输入首选和备用域名服务器的地址。

其它情况下请选择从ISP处动态获取。

注意：如果在访问网站时发生了“Address not found（地址未找到）”错误，那么很可能是域名服务器设置不当。 此时请联系ISP，索要域名服务器地址。

路由器MAC地址

计算机的本地地址在用户网络上是唯一的。 该地址也称为计算机的MAC（介质访问控制）地址。

通常情况下选择“使用缺省MAC地址”即可。

如果ISP要求MAC认证，则请选择使用计算机MAC地址或者使用下面的MAC地址。前一种情况下将把当前计算机的MAC地址伪装成路由器的MAC地址，后一种情况下则需手动输入其它计算机的MAC地址。 MAC地址格式为XX:XX:XX:XX:XX:XX。 在选择了使用下面的MAC地址并且相应地设定了MAC值后，如果再选择使用计算机MAC地址，就会覆盖掉所设定的MAC值。

应用、取消和测试

设置更改完毕后，请单击应用 。

单击取消即可恢复到原来的设置。

单击测试可连接到NETGEAR网站。 如果连接成功，则表明设置正常。此后可单击注销退出这些页面，然后……尽情享受网上冲浪带来的乐趣！

如果连接失败，

  • 则请检查设置，确保所做选择及每一项输入都正确无误。
  • 联系ISP,核实配置信息。
  • 阅读 安装指南中的《故障处理》一节。
  • 阅读 资源CD光盘中的故障处理指南，或者参考手册中《故障处理》一节。
  • 联系NETGEAR技术支持部门。
