地址保留帮助

如果为局域网中的某块PC网卡设置了一个IP保留地址，那么这块网卡每次访问DHCP服务器时，收到的总是同一个IP地址。 需要永久IP设置的服务器均应被分配IP保留地址。

地址保留表

地址保留表中显示了连接在网络中可以进行IP保留的设备。

要保留某个IP地址：

  1. 从 地址保留表中选中待添加计算机旁边的单选按钮。
  2. 如果列表中没有所需计算机，则请单击刷新按钮来更新地址保留表。
  3. 如此时地址保留表仍未列出所需计算机，则请输入待添加计算机的IP地址、MAC地址和设备名称。
  4. 完成后单击添加按钮。

添加或者取消更改

单击添加即可为所选计算机保留一个IP地址。
单击取消即可放弃所有尚未保存的更改。
