智能设置向导帮助

智能设置向导已确定您拥有一个固定IP地址,或者您可能需要特殊的信息才能连接到因特网。 如果因特网服务提供商（ISP）已经为您分配了一个永久不变的IP地址，那么您也就有了一个固定IP。

有没有ISP分配给您的固定或者静态IP地址？

  • 如果ISP已经为您分配了一个永久IP地址，则单击是。 您需要填写那些与固定IP地址设置（IP、子网掩码、网关和域名服务器）相关的必需信息。 ISP应该给了您一份摘要表，上面带有所有必需信息。

  • 如果没有固定IP地址或者不能确定自己的连接类型时，则请单击不确定。 单击不确定后，将返回到基本设置的开始页。在这个页面上，您需要参照ISP指南来查看您的因特网服务提供商是否需要一些特殊设置。 例如，您可能需要输入一个帐户名、域名或使用其它MAC地址。
