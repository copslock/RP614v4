NETGEAR无线帮助

注意： 为保证本产品能够符合当地的政府规定，同时与当地的类似产品实现兼容，必须正确设置工作频道和地区。

选择路由器安放地点，优化无线连接性能

路由器的物理安放地点将显著影响到无线连接的工作距离或范围。 为达到最佳效果，请按以下原则安放路由器：

  • 靠近计算机工作范围的中心部位，
  • 放到高处（比如具有一定高度的架子），
  • 远离潜在干扰源，例如计算机、微波炉和无绳电话等，
  • 天线拉直并直立向上，
  • 远离大块金属表面。

注意：上述方针需切实遵守，否则可能会显著降低性能，甚至无法以无线方式连接到路由器。

无线网络

无线网络标识（SSID）

请在本框中输入一个最多不超过32个字母数字字符的值。 网络中所有无线设备均须分配相同的无线网络标识（SSID）。 缺省SSID为Wireless。但本公司强烈建议您将自己的无线网络标识（SSID）更改为其它值。 本值同样区分大小写。 比如, Wireless和wireless并不相同。

地区

请从下拉列表中选择您所在的地区。 本框显示的是本无线接口的预定工作地区。 除了此处列出的地区之外，在其它地区使用本接入点可能属于非法行为。 如果本列表未列出您所在的国家或者地区，请咨询您当地的政府部门，或者查看我公司网站，了解可以使用哪些频道。

频道

本框确定所用工作频率。 除非发现与附近接入点之间存在干扰问题，否则无线频道无需更改。

无线广播

允许广播无线网络标识（SSID）

无线接入点将广播无线网络标识（SSID）。 这样其它无线客户端即可探测到该设备。

允许从“任意”标识访问

无线接入点将拦截那些对“未指定的SSID”的回应。 没有正确的SSID的无线客户端将被拒绝。

无线网卡访问列表

缺省情况下，无线网络标识配置正确的无线PC网卡均可访问用户的无线网络。 但为了提高安全性，用户可根据PC网卡的MAC地址，仅允许特定的PC网卡访问无线网络。 从无线设置菜单中，单击设置访问列表按钮即可显示“无线访问列表”菜单。

安全加密（WEP）

认证类型

一般情况下本栏可保留缺省值“自动”。 如缺省值无法正常工作，则请选择适当的值—“开放系统”或者“共享密钥”。 具体采用哪种方法，请参考无线网卡说明书。

加密强度

请选择WEP加密程度：

  • 关闭—无数据加密
  • 64位（有时也称为40位）加密
  • 128位加密

安全加密（WEP）密钥

启用WEP后，用户可为自己的网络人工输入或自动生成数据加密密钥。 在网络中所有计算机和接入点上，这些值必须完全相同，如此方可实现连接。

自动生成密钥（密码）

请在“密码”框中输入一个词语或者一组可打印字符，然后单击“生成密钥”按钮，如此即可自动配置WEP密钥。 加密强度设置为64位时，四个密钥框中均会自动显示密钥值。 而加密强度设置为128位时，只有所选的WEP密钥框中才会自动显示密钥值。

手工输入模式

从四个密钥中选择需要使用的一个，然后在所选的密钥框中为用户网络输入匹配的WEP密钥信息。

64位WEP—输入十六进制数字（0-9和A-F之间的任意组合）。

128位WEP—输入二十六位十六进制数字（0-9和A-F之间的任意组合）。

务请单击“应用”，如此方可保存本菜单中的设置。

保存或者取消更改

单击应用，变更即可生效。
单击取消即可返回此前已保存设置。
