UPnP 帮助

通用即插即用（UPnP）帮助网络电器和计算机等设备在需要时接入网络、连接到其它设备。 UPnP设备可自动发现那些来自网络上其它已经登记的UPnP设备的服务。

打开 UPnP

可针对设备自动配置，启用或者禁用UPnP。 UPnP的缺省设置为启用。 如果禁用的话，路由器将不会允许任何设备来自动控制路由器的资源，比如端口转发（映射）。

广告时间间隔

广告时间间隔就是路由器每隔多长时间对其UPnP信息进行一次广告（广播）。 其取值范围介于1到1440分钟之间。 缺省时间间隔为30分钟。 较短的时间间隔将以增大网络流量为代价，确保控制点了解当前的设备状态。 较长的时间间隔会降低设备状态的时效性，但却可以显著降低网络流量。

广告存活时间

广告存活时间的度量，采用的是每个UPnP数据包发出后的跳数（步数）。 跳数就是每个UPnP广告在消失之前允许传播的步数。 跳数取值可介于1到255之间。广告存活时间缺省值为4跳，对多数家庭网络来说应该是足够了。 如果发现未能正确地更新或者达到某些设备，则可能有必要略微增大这一数值。

UPnP 端口映射表

UPnP端口映射表显示了当前正在访问路由器的各台UPnP设备的IP地址，以及每台设备已经打开的端口（内部和外部）。 此外UPnP端口映射表还针对各个IP地址，分别显示所打开的端口的类型，以及该端口是否仍然处于活动状态。

保存、取消或者刷新UPnP端口映射表

单击应用即可将新设置保存到路由器。
单击取消即可放弃所有尚未保存的更改。
单击刷新即可更新端口映射表，同时显示UPnP设备当前打开的活动端口。
