站点限制帮助

如需限制访问因特网上的某些网站，则需设置内容过滤功能。 内容过滤方式共有两种：

  • 对某些域（如www.badstuff.com/xxx）的访问加以限制。或者
  • 对包含某些词汇（如亵渎或明显的黄色读物）的网站加以限制。

当用户试图访问某个被限制的网站时，将会收到消息: "已被 NETGEAR 限制"

要启用站点限制功能:

    单击按时间安排打开站点限制功能，并根据时间安排页面上的设置进行限制。
    单击始终打开站点限制功能，始终进行限制，与时间安排页面无关。

注意：在时间安排页面上，可以设置何时进行关键字和域名限制。

要禁用站点限制功能：

    单击从不，即可关闭站点限制功能。

要限制某些关键字或因特网域：

  1. 选择何时打开关键字限制功能（从不、按时间安排或始终）
  2. 在标有 请在下面输入关键字或域名的框中，输入希望限制的关键字或域名。
  3. 单击添加关键字。
  4. 此时下面的列表中将出现该关键字或者域名。
  5. 继续添加域名和关键字，直到添加完毕。
  6. 完成后单击应用。

限制列表—在"对包含以下关键字或域名的网站进行限制"标题下的列表内，包含有当前要限制项目的列表。

要删除某个关键字或域名：

  1. 从列表中选中该关键字或者域名。
  2. 单击删除关键字。
  3. 继续选择并删除域名和关键字，直到删除完毕。
  4. 单击应用。

要删除全部关键字和域名：

  1. 单击清除列表。
  2. 单击应用。

要允许某台计算机无限制地访问因特网：

  1. 选中允许可信IP地址访问被限制的网站复选框。
  2. 在可信IP地址区域输入此计算机的IP地址。
    只需在最后一个框内输入一个数字即可。
  3. 单击应用。

要允许无限制地访问因特网：

  1. 在关键字限制菜单中选择从不。
  2. 单击应用。
