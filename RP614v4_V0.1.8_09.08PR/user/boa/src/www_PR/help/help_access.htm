无线网卡访问列表帮助

缺省情况下，无线网络标识配置正确的无线PC网卡均可访问用户的无线网络。 但为了提高安全性，用户可根据PC网卡的MAC地址，仅允许特定的PC网卡访问无线网络。

无线网卡访问设置页面中显示有一张无线PC网卡列表，表中的网卡都允许连接到本路由器（根据其MAC地址）。 不过要访问无线网络，还必须在无线设置页面中,正确地配置这些无线PC网卡的无线网络标识和WEP设置。

打开无线访问控制

    1. 单击打开无线访问控制复选框，即可根据无线PC网卡的MAC地址对无线PC网卡进行限制。
    2. 单击应用按钮即可保存更改并返回到无线设置页面。

注意：启用了打开无线访问控制后“访问控制列表”为空时，所有无线PC网卡均无法连接到用户无线网络。

设置“访问控制列表”：

    1. 单击添加按钮，进入访问设置菜单。 为便于用户操作，该菜单中显示有一张列表，其中列出了当前处于活动状态的无线网卡及其以太网MAC地址。
    2. 列表中出现了所需的PC网卡时，可单击该PC网卡的单选按钮，捕获其MAC地址；否则可手工输入待授权PC网卡的MAC地址。 无线设备的底部一般都标有其MAC地址。
    3. 如果没有出现“设备名称”，则可以为所添加的PC网卡键入一个描述性名称。
    4. MAC地址输入完毕之后，单击添加按钮即可返回到“无线访问列表”菜单。
    注意：各无线PC网卡均需重复步骤1到4。
    5. 单击打开无线访问控制复选框，启用“访问控制”。
    6. 单击应用按钮即可保存更改并返回到无线设置页面。

编辑“访问控制列表”中的PC网卡：

    1. 单击待编辑无线PC网卡的单选按钮。
    2. 单击编辑按钮。
    3. 编辑“设备名称”或MAC地址。
    4. 完成后单击接受按钮。
    5. 单击应用按钮即可保存更改并返回到无线设置页面。

从“访问控制列表”删除PC网卡：

    1. 单击待删除无线PC网卡的单选按钮。
    2. 单击删除按钮。
    3. 单击应用按钮即可保存更改并返回到无线设置页面。

保存或者取消更改

设置更改完毕后，请单击应用 。
单击取消即可返回此前已保存设置。
