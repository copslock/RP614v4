静态路由帮助

静态路由将为路由器提供一些路由器无法通过其它方式自动了解的信息。 局域网上禁用了RIP时就会发生这种情况。 （参见局域网IP设置页面。） 本张表格中将显示所有已定义的静态路由。 如需增加或者删除路由，请在静态路由表下方区域内操作。

要设置一条静态路由：

  1. 单击添加按钮。
  2. 在路由表下方的路由名称框内键入该静态路由的路由名称。
    （路由名称仅供标识路由之用。）
  3. 如果需要将访问限制在局域网内部，则请选择私有。
  4. 选择活动，另本条路由生效。
  5. 键入最终目的地的目的IP地址。
  6. 键入该目的地的IP子网掩码。
    如本路由的最终目的地为单一主机，则请键入255.255.255.255。
  7. 键入网关IP地址，该地址必须是同一网段中的某台路由器。
  8. 键入一个2到15之间的数值作为 度量值。
    度量值代表的是用户网络上其它路由器的数量。 通常情况下度量值设置为2或者3可以达到最佳性能。不过如果本路由为直接连接，则请将度量值设置为2。
  9. 单击应用之后即可将本条静态路由添加到路由表。

要编辑某条静态路由：

  1. 单击待编辑路由旁边的单选按钮。
  2. 更新信息。
  3. 完成后单击应用。

要删除某条静态路由：

  1. 单击待删除路由旁边的单选按钮。
  2. 单击删除。
  3. 单击应用即可将该条静态路由从路由表中删除。
