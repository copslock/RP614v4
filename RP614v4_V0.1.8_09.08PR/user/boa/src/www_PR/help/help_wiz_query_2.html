智能设置向导帮助

智能设置向导检测到了一个业已存在的因特网活动连接。 目前看来这个因特网连接工作正常，可能无需继续运行本智能设置向导。 如果决定继续运行本智能设置向导，则有可能更改当前的设置。 建议您在继续运行本智能设置向导之前，对当前设置进行备份。

操作步骤如下：

  1. 如果您需要继续运行本智能设置向导，那么请单击是。
  2. 如需退出本智能设置向导并保持当前的因特网连接，则请单击否。

要对设置进行备份：

  1. 如需退出本智能设置向导并保持当前的因特网连接，则请单击否。
  2. 在导航栏中单击备份设置，进入“备份菜单”。
  3. 保存了设置之后即可重新运行本智能设置向导。
