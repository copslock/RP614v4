电子邮件帮助

LAN（局域网）上有人试图访问被限制站点时可发送报警。 而日志中则列出了所有已访问的URL地址。 如果希望以电子邮件的方式向您发送报警和日志，那么请填写本表格中的设置。

无论何种情况，您均可查看日志页面，据此来人工检查日志。 如果不需要接收电子邮件，那么请不要填写本表各框。

要通过电子邮件接收报警和日志：

  1. 选中打开电子邮件通知复选框。
  2. 在邮件发送服务器框中，键入ISP的SMTP邮件发送服务器（例如smtp.myISP.com）。
    如果不填本框，则不会向您发送报警或者日志。
    技巧：在设置电子邮件程序时用到过这些信息。 如果记不住，则可查看电子邮件程序中的设置。
  3. 在电子邮件地址框中键入一个电子邮件地址，让报警和日志发送到您或者别人那里。
    请使用完整的电子邮件地址（例如Jackie@myISP.com）。
  4. 如果需要在网络上有人试图连接到某个被限制URL时立即发送电子邮件报警，则请选中立即发送报警复选框。
  5. 如果您不需要发送日志，则请在按照本时间安排发送日志区域中的第一个列表中选择无。
    而
    如果需要发送日志的话，则请选择其它选项。
    • 如果选择了每周，则请选择星期几。
    • 如果选择了每周或者每天，则请选择在一天中的哪一时间发送电子邮件。
  6. 单击应用，让变更生效。

时区。

选择本地时区。 只有选择了本项之后，时间安排才能正常发挥功能。

如果所在地区需要的话，请选中“按夏时制调整”。
