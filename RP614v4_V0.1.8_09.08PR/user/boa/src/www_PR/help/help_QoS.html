基于应用的QoS帮助

基于应用的QoS是可用于因特网应用的一种先进功能。 对基于应用的QoS来说，用户能指定一个目的端口号，并将其关联到较高的准入优先级。

基于应用的QoS

对于下拉菜单中已经存在的服务、应用或游戏，其端口号应通过单击添加自定义服务按钮来指定。

端口分配

您最多可针对因特网服务、应用或者程序进行10个不同的端口分配。 在“服务名称列表”中可以选择一个服务、应用或者游戏。 如果在所有列表中都没发现所需的项目，则请联系软件或者游戏开发人员，查找需要使用的正确端口号。

因特网服务

在开始之前，首先应确定即将提供的服务的类型。

要为某因特网服务设置基于应用的QoS：

  1. 从“服务名称”列表中选择一项需要使用的服务。
  2. 单击添加。

因特网游戏或者应用

开始前,首先要了解待配置的是哪种服务、应用或者游戏。

要为某因特网应用设置基于应用的QoS：

  1. 从相关的某个列表中选择需要使用的因特网应用或者游戏。
    此时起始端口和结束端口框中均会填充相应值。
    注意： 如果在列表中找不到所需的游戏或者应用，则请单击添加自定义服务按钮，然后键入服务名称、起始端口和结束端口等信息。
  2. 单击添加。

要设置其它基于应用的QoS，例如Hexen II或KALI：

  1. 单击添加自定义服务按钮。
  2. 在服务名称框中键入服务名称。
  3. 在起始端口框中键入起始端口号。
    对于此类游戏来说，请使用默认列表中给出的端口号，每增加一台计算机，端口号加1。 举例来说，假设有一台计算机被配置成了采用端口26900来运行Hexen II，那么第二台计算机的端口号就应该是26901，第三台计算机的应为26902。
  4. 在结束端口框中键入相同端口号。
  5. 单击应用。

要修改某项服务：

  1. 选中表内该项服务的单选按钮。
  2. 单击编辑服务。
  3. 从服务名称、起始端口和结束端口各框中修改信息。
  4. 单击应用。

要删除某项服务：

  1. 选中表内该项服务的单选按钮。
  2. 单击删除服务。
